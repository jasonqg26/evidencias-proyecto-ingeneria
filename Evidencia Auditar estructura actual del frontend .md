# Evidencia de Tarea - Sprint

**Nombre:** Jason Quesada Gomez
**Sprint:** 2
**Tarea:** Auditar estructura actual del frontend
**Fecha:** 1/06/2026

## Trabajo realizado

- Revisar carpetas, componentes, páginas y módulos existentes.

- Detectar inconsistencias y malas prácticas.


## Hallazgos identificados

* Componentes con más de 1000 líneas de código.
* Responsabilidades mezcladas dentro de un mismo componente.
* Lógica de negocio y presentación acopladas.
* Reutilización limitada de componentes.
* Estructura de carpetas inconsistente entre módulos.
* Dificultad para realizar mantenimiento y pruebas.

## Recomendaciones

* Dividir componentes grandes en componentes más pequeños y reutilizables.
* Separar la lógica de negocio mediante hooks personalizados.
* Estandarizar la estructura de carpetas del proyecto.
* Aplicar principios de responsabilidad única en los componentes.
* Mejorar la reutilización de componentes compartidos.

## Resultado

Se generó un diagnóstico del estado actual del frontend y se identificaron oportunidades de mejora para aumentar la mantenibilidad, escalabilidad y legibilidad del código.
